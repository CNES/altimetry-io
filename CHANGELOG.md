# Changelog

All notable changes to this project will be documented in this file.

---

## [0.2.0] - 2026-03-05

### Added
- Introduced a `concat` parameter to control the output format of queries (either a complete dataset or a list of datasets).

### Fixed
- Correctly handle queries on file collections that are empty or contain an unexpected data type.

---

## [0.1.4] - 2026-02-10

### Added
- Added this changelog file.

### Changed
- Improved the README documentation.

---

## [0.1.3] - 2026-02-10

### Added
- Initial release.
- Support for reading the following local file collections:
  - SWOT LR SSH L2 / L3
  - SWOT LR Wind Wave L3
  - Nadir L2 / L3 data
- Support for reading the ZCollection data format for internal CNES HPC use.
